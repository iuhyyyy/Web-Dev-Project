# Predikt

A community-driven prediction market platform where users forecast real-world events using fixed odds.

## Pages

| File | Description |
|------|-------------|
| `index.html` | Home — hero, trending markets, how it works |
| `markets.html` | Browse all open markets with live search/filter |
| `market.html` | Single market detail — odds, stats, place prediction |
| `leaderboard.html` | Top predictors ranked by accuracy score |
| `profile.html` | Sample user profile with prediction history |
| `about.html` | How Predikt works, team, mission |
| `contact.html` | Contact form with JS validation |
| `faq.html` | Frequently asked questions |

## Tech Stack

- HTML5, CSS3, JavaScript 
- Bootstrap 5.3
- Custom CSS variables for theming
- No frameworks or build tools — plain files

## Interactive Features

1. **Live search & filter** on `markets.html` — filters market cards in real time by keyword and category
2. **Contact form validation** on `contact.html` — validates name, email, and message before submission
